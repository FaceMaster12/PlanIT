# PlanIT - Django To-Do List App

PlanIT is a simple, interactive, and user-friendly **To-Do List web application** built with **Django**.  
It allows users to create, update, and delete tasks, mark them as complete, and manage their daily activities efficiently.  

---

## **Features**

- User authentication: register, login, and logout
- Create, update, and delete tasks
- Mark tasks as completed
- Tasks are user-specific (each user sees only their tasks)
- Admin panel to manage all tasks and users
- Responsive front-end with CSS and interactive JavaScript
- Ready to deploy on free hosting platforms like PythonAnywhere or Render

---

## **Tech Stack**

- **Backend:** Django, Python  
- **Database:** SQLite (default) / MySQL (optional)  
- **Front-end:** HTML, CSS, JavaScript  
- **Version Control:** Git / GitHub
